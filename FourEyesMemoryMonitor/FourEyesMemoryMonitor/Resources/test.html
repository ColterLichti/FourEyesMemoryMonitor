FourEyes Memory Monitor

Author: Colter Lichti

Instructions

To use four eyes simply open the application. If it is your first time using FourEyes you should see the Control Panel. If you don't look in windows tray (to the left of the clock) for the glasses icon and double click it. You can also right click the icon and select "Show Control Panel".


From the Control Panel you can:
  • Set the Process Name
  • Set the Memory Threshold
  • Set the Opacity Value
  • Set the Poll Interval
